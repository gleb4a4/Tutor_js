Выражение - это набор переменных, констант, операторов. У любого выражения есть свое значение - результат вычисления выражения. Значение может быть числом, строкой или логическим значением .

Операторы присваивания

ОператорПримерОписание
+=x += yx = x + y
-=x -= yx = x - y
*=x *= yx = x * y
/=x /= yx = x / y
%=x %= yx = x % y

Операции вычисления

  • + - сложение (например, А = В + С)
  • - - вычитание (например, А = В - С).
  • * - умножение (например, А = В * С).
  • / - деление (например, А = В / С).
  • % - деление по модулю (например, А = В % С).

Инкеремент && Декремент

  • ++ - инкремент, увеличивает значение переменной на 1 (например, i++ ).

Переменной i будет присвоено значение n, а после этого переменная i будет
увеличена на 1 (значение 2). Если же ++ указать до i (а не после него), то
сначала переменная i будет увеличена на 1, а потом уже будет присвоено
новое значение переменной i: примеры в js файлике

  • -- - инкремент, уменьшает значение переменной на 1 (например, i-- ).

Переменной i будет присвоено значение n, а после этого переменная i будет
уменьшена на 1 (значение 0). Если же -- указать до i (а не после него), то
сначала переменная i будет уменьшена на 1, а потом уже будет присвоено

Логические операторы

  • ! - унарная операция отрицания (NOT).
  • && - бинарная операция И (AND), истинна, когда оба операнда истинны.
  • ||- бинарная операция ИЛИ (OR), истинна, когда один из операндов равен true.
